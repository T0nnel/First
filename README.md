# ToolHub - Next.js SEO-Optimized Tools Platform

Professional free online tools platform with 20+ tools. Built with Next.js 14, optimized for SEO and traffic.

## Live Demo
[Your deployed site will be here]

## Features
- 8+ working tools (Word Counter, JSON Formatter, QR Generator, etc.)
- Individual SEO-optimized pages for each tool
- Auto-generated sitemap
- Google AdSense ready
- Mobile responsive
- Fast performance

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Or use Vercel dashboard: https://vercel.com

## Monetization

1. Apply for Google AdSense
2. Add your publisher ID in src/app/layout.tsx
3. Replace ad placeholder divs with ad codes

## SEO

- Each tool has unique URL and metadata
- Sitemap auto-generated at /sitemap.xml
- Submit to Google Search Console after deployment

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React
